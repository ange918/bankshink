## Format des Clés

Toutes les clés suivent le format `section_element_type` :
- **section** : La zone du site (nav, hero, product, tier, etc.)
- **element** : L'élément spécifique (title, description, button, etc.)
- **type** : Le type optionnel (1, 2, 3 pour les variantes)

3. Le système multilingue est déjà configuré dans `js/translations.js`
